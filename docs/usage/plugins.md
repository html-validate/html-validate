# Plugins

Plugins extend the functionality of HTML-validate:

- Custom rules.
- Custom metadata.
- Custom configuration presets.
- Transformers.

## List of third-party plugins

- [html-validate-ember]: transforms Ember templates (`.gts`, `.gjs` and `.hbs`) and applies recommended configuration.

[html-validate-ember]: https://github.com/johanrd/html-validate-ember
